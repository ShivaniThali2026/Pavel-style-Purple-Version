# Pavel Style Purple Version

Combined Shivani Thali portfolio version using:

- Layout and section structure from `Pavel-style-Version-3`
- Purple visual direction and profile photo from `shivani-portfolio-creative`

The site is configured for GitHub Pages at:

```text
https://shivanithali2026.github.io/Pavel-style-Purple-Version/
```

## Local Development

```bash
npm install
npm run dev
```

Open:

```text
http://localhost:3000/Pavel-style-Purple-Version
```

## Build

```bash
npm run build
```

Next exports the static site to `out/`. Copy `out/` to `docs/` before publishing with GitHub Pages from the `docs` folder.
